# Codeforces Modern Design

A modern UI redesign for Codeforces platform [by funcdfs](https://github.com/funcdfs)

![Preview](https://github.com/funcdfs/change-Codeforces-UI/assets/56250171/2ac119a3-668e-4aed-aba7-6d537a297fce)

## Installation

1. Install a userscript manager for your browser (like Tampermonkey)
2. Install the script from [Greasy Fork](https://greasyfork.org/scripts/470204-change-codeforces-ui)

## Features

- Problem navigation bar showing other problems from the same contest
- Luogu problem links integration (for Chinese users)
- Friend submission history in top bar
- Cleaner interface:
  - Removed footer sponsor content
  - Repositioned top bar elements
  - Username moved to right side
  - Problem metadata (time/memory limits) moved to less prominent position
  - Modern UI elements (rounded corners, shadows, scrollbars, etc.)
